# Deepfake Detection with Hybrid CNN + Transformer

This project implements a deepfake detection system using a hybrid architecture combining a ResNet50 backbone (CNN) for local feature extraction and a Transformer Encoder for global feature modeling.

## Project Structure

```
├── data_raw/           # Place your raw 'real' and 'fake' images here
│   ├── real/
│   └── fake/
├── data_processed/     # Generated by preprocess.py
├── src/
│   ├── preprocess.py   # Data preprocessing pipeline
│   ├── model.py        # Hybrid Model architecture
│   ├── train.py        # Training script
│   ├── evaluate.py     # Evaluation metrics
│   ├── predict.py      # Inference logic
│   └── app.py          # Streamlit Web App
├── requirements.txt    # Dependencies
└── README.md           # This file
```

## Setup

1.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Prepare Data**:
    - Create `data_raw/real` and `data_raw/fake` directories.
    - Add your dataset images to these folders.

## pipeline

### 1. Preprocessing
Run the preprocessing script to detect faces, resize, normalize, and augment the data.

```bash
python src/preprocess.py
```
This will create processed images in `data_processed/`.

### 2. Training
Train the model. It uses early stopping and saves the best model as `best_model.pth`.

```bash
python src/train.py
```

### 3. Evaluation
Evaluate the trained model on the validation set.

```bash
python src/evaluate.py
```

### 4. Web Application (Inference)
Run the Streamlit app to test the model with your own images.

```bash
streamlit run src/app.py
```

## Model Architecture
- **Backbone**: ResNet50 (Pretrained on ImageNet)
- **Global Context**: Transformer Encoder (2 Layers, 8 Heads)
- **Classification**: Binary (Real vs Fake) w/ Sigmoid activation
